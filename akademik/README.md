1. Clone repo ini:
   ```bash
   git clone https://github.com/alpaenf/akademik_livewire.git
   cd akademik_livewire
   ```

   ```bash
   composer install
   npm install
   ```

   ```bash
   cp .env.example .env
   ```

   ```bash
   php artisan migrate
   npm run dev
   ```
5. Jalankan server:
